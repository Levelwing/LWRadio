Customize with Sass

You can use these Sass variables to customize the icons:

$iconic-primary-fill: $primary-color; $iconic-primary-stroke: $primary-color; $iconic-accent-fill: $iconic-primary-fill; $iconic-accent-stroke: $iconic-accent-fill;
